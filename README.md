# myFirstGitTest
a test for git

##### 1 指定当前目录为git仓库

```shell
git init
```


##### 2 绑定本地与github仓库

```shell
  git remote add origin https://github.com/wobukewu/git-test.git
```

##### 3 提交

```shell
  git add *		将指定文件纳入git版本控制
  git commit -m '提交描述descrp' 以m分支提交
```

##### 4 克隆

```shell
git clone <repo><directory>  克隆<repo>地址的仓库到本地目录<directory>
```
